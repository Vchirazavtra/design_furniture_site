ИНСТРУКЦИЯ К ПРОЕКТУ "DESIGN FURNITURE"

1. ОБЩЕЕ ОПИСАНИЕ
Проект представляет собой веб-приложение интернет-магазина мебели.
Основной функционал:
- Регистрация и авторизация пользователей
- Просмотр каталога товаров
- Страница товара
- Корзина
- Оформление заказов
- Профиль пользователя
- Административная панель

2. УСТАНОВКА И ЗАПУСК

2.1 Установка зависимостей:
pip install flask werkzeug pymysql

2.2 Создание базы данных:
CREATE DATABASE furniture;

2.3 Импорт базы:
mysql -u root -p furniture < dump-design_furniture_catalog.sql

2.4 Настройка подключения (db.py):
Указать host, user, password и database.

2.5 Запуск:
python app.py

3. СТРУКТУРА ПРИЛОЖЕНИЯ

app.py — основной файл приложения
templates/ — HTML шаблоны
static/ — стили, изображения и скрипты
database — MySQL база данных

4. ФУНКЦИОНАЛ

4.1 Пользователи:
- Регистрация (/register)
- Авторизация (/login)
- Выход (/logout)

4.2 Каталог:
- /home — главная страница
- /catalog — список товаров с фильтрами
- /product/<id> — страница товара

4.3 Корзина:
- /cart — просмотр корзины
- /cart/add/<id> — добавить товар
- /cart/update_item — изменить количество
- /cart/remove_item — удалить товар

4.4 Заказы:
- /cart/checkout — оформление заказа

4.5 Профиль:
- /profile — список заказов

4.6 Админка:
- /admin — управление товарами

5. БАЗА ДАННЫХ

Основные таблицы:
